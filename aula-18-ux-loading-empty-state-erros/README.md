## O Desafio Prático
Um aplicativo que funciona é bom. Um aplicativo que protege o usuário do medo de travamento é Premium. 
O que acontece na sua tela durante os 200 milissegundos que o motor SQLite leva para achar uma lista de 500 produtos? Fica tudo branco? O botão parece que não clicou e o cliente aperta 5 vezes quebrando seu sistema? E se não houver NENHUM produto na tabela? 

A arte do **User Experience (UX)** lida não com o código que "dá certo", mas como nós comunicamos ao cliente humano sobre o Fluxo do Tempo ou Quebras Severas da máquina.
