## O Desafio Prático
Se você colocar um texto e uma foto do Neymar num aplicativo, quem decide se eles ficarão um do lado do outro, em fila indiana ou no meio da tela pulando de um lado para o outro?
A resposta é o motor de gravidade embutido no React Native: O **Flexbox**.

Nesta Aula, vamos parar de desenhar as coisas diretamente no "chão" da nossa cozinha virtual, e faremos gavetas isoladas (Padrão de Componentização) para abrigar nossa Imagem no aplicativo **StickerSmash**, usando o mais puro poder de renderização nativa!
