# Tutoriais – PAM I

Guia dos tutoriais e atividades por aula. Siga em ordem para aproveitar o curso.

---

## Visão geral

Cada aula possui três materiais (todos linkados no **índice da aula** – README da pasta):

- **apresentacao.md** – Slides da aula em Markdown.
- **tutorial.md** – Passo a passo para fazer um exemplo em sala ou em casa (comandos e conceitos).
- **atividade.md** – Exercício para entregar, baseado no tutorial mas com variação para fixar o conteúdo.

Use o link **Índice** de cada linha para acessar Apresentação, Tutorial e Atividade.
