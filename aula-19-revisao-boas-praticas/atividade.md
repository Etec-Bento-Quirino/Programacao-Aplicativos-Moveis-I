## O Desafio: Construir o Contrato de Trabalho Inicial

Você irá fazer O Design do seu próprio `README.md` Base para O Seu próprio projeto em gestação, que será consolidado e entregue como PROJETO FINAL do Semestre na Aula que segue. Crie este doc dentro da Raiz do seu Expo.

O que precisa obrigatoriamente estar estruturado Bonito Dentro dele?

1. **Título Titã e Objetivo do App:** Um Parágrafo Vendedor! ("Este é um app feito pra contar pinguins", "Este App escaneia metas semanais", etc)
2. **Caixinhas de Telas Explicadas:** Pontue quantas telas seu arquivo terá e o que o Componente Raiz exibe pro Cidadão. (Ex: "Tela de Login e Tela de Banco").
3. **Pilar do SQLite Relacional:** Escreva os Códigos SQL Brutos Base Criativos de Tabelas e Colunas Relacionais do Seu Arquivo C++. Eu quero saber o DDL do seu Banco Sem Ver seu Código Fonte Real!
4. **Manual Brutal de Inicialização:** Ensine a pessoa da Rua que não entende de Expo a dar um `npm install` no seu Clone e a rodar via Local CLI Server pro celular Ligar! Exija que o cara faça o teste nele.
