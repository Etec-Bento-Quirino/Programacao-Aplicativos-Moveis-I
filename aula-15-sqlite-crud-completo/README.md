## O Desafio Prático
Temos os tijolos (SQLite Local), temos a tinta (React Native Engine), e temos o Pintor (A Mente, useState).
Hoje nós alcançaremos a glória conectando e operando os 4 Pilares da Engenharia de Software global: CREATE, READ, UPDATE, DELETE (O Famoso CRUD). Faremos a tabela "conversar" com a lista da tela dinamicamente em ambas as direções.
