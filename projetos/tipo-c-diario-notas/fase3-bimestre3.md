# Tipo C – Fase 3 (Bimestre 3)

**Sugestão de execução:** quinzena 20 (13/10/2026 a 14/10/2026).

## Objetivo

SQLite: tabelas `categorias` e `notas` (id_categoria, titulo, texto, caminho_foto, data). CRUD completo; listagem por categoria.

## Requisitos

1. **Tabelas:** categorias (id, nome); notas (id, id_categoria, titulo, texto, caminho_foto, data).
2. **CRUD:** criar/editar/excluir categoria e nota; listar notas por categoria.
3. **Formulário nota:** seleção de categoria; título, texto; opcional foto (guardar caminho ou string).
4. **Telas:** lista categorias → lista notas da categoria → detalhe/edição nota.

## Conteúdos cobrados

Aulas 14, 15, 16, 17.

## Critérios de avaliação

SQLite com duas tabelas; CRUD; listagem por categoria; foto opcional.
