## O Desafio Prático
Se a interface fosse um corpo humano, as Views seriam o esqueleto e o StyleSheet a pele. Mas e a mente? Quem é que percebe que o botão foi clicado e envia estímulos para modificar o Layout? Quem é que decide a hora exata de invocar o sistema da internet sem travar o usuário?

Bem-vindos à Fase 3 do nosso curso! Hoje nós destrancamos a maior criação da Engenharia do React: Os React Hooks (`useState` e `useEffect`), as pontes lógicas que fazem sua aplicação deixar de ser um desenho para se tornar uma Inteligência capaz de raciocinar com memória.
