## O Desafio Prático
Um aplicativo de uma única página raramente cumpre as demandas do mercado. O WhatsApp tem a lista de chats, a tela de status e a tela de configurações. Como trocamos a tela sem "recarregar a página" inteira, garantindo fluidez nativa?

Nesta Quinzena entraremos no universo vital das Rotas. Aboliremos o antigo modelo verboso e confuso do React Navigation padrão e usaremos o moderníssimo **Expo Router** baseado em Sistemas de Arquivo (File-Based Routing).
